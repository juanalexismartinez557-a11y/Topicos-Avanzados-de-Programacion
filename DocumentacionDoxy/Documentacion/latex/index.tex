\chapter{Proyecto de Análisis de Complejidad Algorítmica }
\hypertarget{index}{}\label{index}\index{Proyecto de Análisis de Complejidad Algorítmica@{Proyecto de Análisis de Complejidad Algorítmica}}
\hypertarget{index_intro}{}\doxysection{\texorpdfstring{Introducción}{Introducción}}\label{index_intro}
Este proyecto implementa algoritmos de búsqueda y ordenamiento, junto con un generador de números aleatorios configurable.

Se analiza la complejidad computacional de cada algoritmo y se mide el tiempo de ejecución utilizando la librería $<$chrono$>$.\hypertarget{index_funcionalidades}{}\doxysection{\texorpdfstring{Funcionalidades}{Funcionalidades}}\label{index_funcionalidades}

\begin{DoxyItemize}
\item Generación de números aleatorios (con y sin repetición)
\item Búsqueda Secuencial
\item Búsqueda Binaria
\item Ordenamiento Burbuja
\item Ordenamiento Selección
\item Ordenamiento Inserción
\item Quick\+Sort
\item Merge\+Sort
\end{DoxyItemize}
