Documentación del Proyecto

Proyecto: Aplicación de Procesamiento de Imágenes con OpenCV

Lenguaje: C++/CLI

Interfaz gráfica: WinForms

Integrantes: Juan Alexis Moran Gómez, Victor Eduardo Reyes Gonzales, Luis Diego Aguilar Rodríguez

1. Introducción

Este proyecto consiste en el desarrollo de una aplicación con interfaz gráfica que permite cargar imágenes desde el sistema o utilizar la cámara del equipo para aplicar distintos filtros de procesamiento digital de imágenes. La aplicación fue desarrollada utilizando C++/CLI con WinForms y la biblioteca OpenCV.

2. Tecnologías Utilizadas

  • C++/CLI
  • Windows Forms (WinForms)
  • OpenCV
  • Visual Studio

3. Estructura del Proyecto

El proyecto se encuentra organizado en distintos archivos que permiten separar la lógica de la interfaz gráfica y el procesamiento de imágenes:

  • MyForm.h – Contiene la interfaz gráfica y los eventos de los botones.
  • ImageProcessor.h / ImageProcessor.cpp – Implementan los filtros de OpenCV.
  • main.cpp – Punto de entrada del programa.
  • .vcxproj – Archivo de configuración del proyecto.

4. Interfaz Gráfica

La interfaz gráfica fue desarrollada en WinForms y cuenta con los siguientes elementos:

  • Botón para cargar una imagen desde el equipo.
  • Botón para activar la cámara.
  • Lista desplegable (ComboBox) para seleccionar el filtro.
  • TrackBars (sliders) para modificar valores como el umbral.

Los cambios realizados mediante los controles se reflejan en tiempo real sobre la imagen.

5. Filtros Implementados

Escala de Grises

Convierte la imagen a un solo canal de intensidad, reduciendo la información de color y facilitando el análisis de la imagen.

Binarización

Transforma la imagen en blanco y negro utilizando un valor de umbral. Se implementan dos variantes: blanco/negro y negro/blanco.

Canny

Algoritmo de detección de bordes que utiliza gradientes para identificar contornos en la imagen.

Sobel

Operador que permite detectar bordes en dirección horizontal y vertical mediante derivadas.

Separación de Canales RGB

Permite visualizar de forma independiente los canales rojo, verde y azul de una imagen.

Rotación y Zoom

Transformaciones geométricas aplicadas a la imagen para cambiar su orientación o escala.

6. Uso de TrackBars

Los TrackBars permiten modificar dinámicamente los valores asociados a los filtros, como el umbral en la binarización o los parámetros en los detectores de bordes, reflejando los cambios de forma inmediata en la imagen mostrada.

7. Proceso de Desarrollo

  1. Configuración del entorno de desarrollo con OpenCV.
  2. Creación de la interfaz gráfica en WinForms.
  3. Implementación de botones y eventos.
  4. Integración de filtros de OpenCV.
  5. Pruebas y ajustes de parámetros mediante TrackBars.

8. Conclusión

Este proyecto permitió aplicar conocimientos de procesamiento digital de imágenes, programación en C++ y desarrollo de interfaces gráficas, fortaleciendo el trabajo en equipo y el uso de bibliotecas externas como OpenCV.
